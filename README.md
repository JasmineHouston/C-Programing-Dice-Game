# C++ Programing-Dice-Game
This project is a C++ programing-based dice game that simulates two players rolling dice and competing for the highest score.
Random number generation (rand (), srand ()
Conditional statements (win/lose/tie logic)
Functions (modular dice rolling logic)
Basic input/output operations 
Each player rolls two dice (random numbers 1-6). The scores are calculated by summing the dice values.
The program compares scores and displayes the winner.
